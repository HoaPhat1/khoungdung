Nội dung Sub Tab 2

Đây là nội dung của sub tab 2 trong tab IOS.

Hướng dẫn chặn thu hồi chứng chỉ Apple

Các ứng dụng cần thiết:

- Temp Mail.
- Document.
- Các file Mobileconfig.

Các bước thực hiện:

- Tải các ứng dụng Temp Mail, Document.
- Tải về file Mobileconfig tại đây.
- Tải về file rules tại đây.
- Truy cập trang adguard-dns.io. Chọn Dùng thử AdGuard DNS.
- Mở Temp Mail lên chọn mail rồi điền vào mục email trong trang Adguard-dns, tạo mật khẩu sau đó ấn vào đăng nhập.
- Chọn Connect to device.
- Trong mục Device name đặt tên bât kỳ. Ví dụ: Chặn 1.
- Ấn next.
- Ấn Home.

- Kéo xuống dưới cùng ấn chọn vào mục User rules.

- Ấn chọn Import rồi chọn file rules.txt vừa tải về.

- Ấn vào logo Adguard DNS.

- Ấn chọn Settings.

- Kéo xuống dưới cùng chọn copy DNS.

- Mở file Chặn thu hồi 1 lên.
- Làm tương tự để tạo ra được 5 file chặn thu hồi chứng chỉ dạng .mobileconfig.
